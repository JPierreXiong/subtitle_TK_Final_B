## 🔧 如果再次遇到启动问题

### 快速修复命令
```powershell
# 停止所有 Node 进程
Get-Process -Name node | Stop-Process -Force

# 清理锁文件
Remove-Item -Path ".next\dev\lock" -Force -ErrorAction SilentlyContinue

# 重新启动
pnpm dev
```

### 彻底清理（如果问题持续）
```powershell
# 删除整个 .next 目录
Remove-Item -Path ".next" -Recurse -Force

# 重新启动（会重新编译）
pnpm dev
```
