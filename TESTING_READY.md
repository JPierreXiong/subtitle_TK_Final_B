## 🧪 快速测试步骤

### 1. 访问应用
打开浏览器访问：**http://localhost:3000**

### 2. 测试视频提取
1. 输入一个测试 URL（YouTube 或 TikTok）
2. 选择输出类型（字幕或视频）
3. 点击"提取"按钮
4. 观察：
   - ✅ 应该立即返回响应（202 Accepted）
   - ✅ 显示任务 ID
   - ✅ 状态开始更新

### 3. 检查控制台
- 打开浏览器开发者工具（F12）
- 查看 Console 标签的日志
- 查看 Network 标签的 API 请求

### 4. 检查后端日志
- 查看终端/控制台输出
- 查找 QStash 相关日志（如果配置了）
- 查找 Worker 处理日志

## 🔍 如果遇到问题

### 服务器无法访问
1. 检查端口 3000 是否被占用
2. 查看终端是否有错误信息
3. 尝试重新启动服务器

### 功能不工作
1. 检查浏览器控制台错误
2. 检查网络请求是否成功
3. 检查后端日志
4. 检查环境变量配置
